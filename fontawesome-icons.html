Font Awesome 4.7.0 图标库

关于 Font Awesome

Font Awesome 是一套流行的图标字体库，包含了大量常用图标，可通过 CSS 类名直接使用。

本页面展示了 Font Awesome 4.7.0 版本的常用图标，点击图标可复制使用代码。

使用方法：在 HTML 中添加 <i class="fa fa-icon-name"></i> 即可显示对应图标。
